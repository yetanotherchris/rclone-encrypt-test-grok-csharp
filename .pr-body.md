## Summary
- Implements a cross-platform, single-file, trimmed .NET 10 CLI that is compatible with rclone crypt (standard name encryption + file content).
- Encryption: scrypt (N=16384,r=8,p=1), EME+AES-256 for filenames (base32 or base64), NaCl SecretBox (XSalsa20+Poly1305) for file content with 64KiB chunks.
- CLI matches request: `-i/--input-file`, `-o/--output-file`, prompts for password + optional salt, `--password` (with security warning), `--encoding base32|base64`, `--filename-only`.
- Tag-based releases: `.github/workflows/build-release.yml` cross-compiles for win/mac/linux amd64/arm64, publishes GitHub Release, updates Scoop manifest + Homebrew formula (matching the rclone-web pattern).
- Tests cover encrypt/decrypt (with/without salt), base64 encoding, and --password usage. Filename roundtrips and known vectors for empty-password/default-salt.

## Security
- `--password` prints a warning and recommends `RCLONE_CRYPT_PASSWORD` or the interactive prompt (no echo). After use, clear shell history.
